# Prompts

## Available tools

The PCC Agent toolkit provides the following tools:

**Healthcare Data Management**

- `get_patient_data`: Get patient data from PCC for the app "Example"
- `get_activated_vendor_apps`: Get activated vendor apps from PCC for the org "123333"
- `get_facility_data`: Get the facilities for the org from PCC
- `get_org_info`: Get the org information for the app "Example"

> **Example Prompt**:
> List the activated vendor apps
